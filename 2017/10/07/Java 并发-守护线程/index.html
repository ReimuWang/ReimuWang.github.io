Java 并发-守护线程

发表于 2017-10-07 更新于 2018-03-29 分类于 Java 并发

守护线程(Daemon)：完成系统性服务。例如垃圾回收线程，JIT线程。

用户线程：完成应用程序的业务操作。

当用户线程全部结束时，也就意味着程序员指定的需求都已完成了。守护线程要守护的对象已经不存在了，守护线程也会结束。整个应用程序也就自然结束了。
